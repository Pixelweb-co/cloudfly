### 2️⃣ AI Hand‑off
| Método | Ruta | Descripción | Request Body | Response | Código HTTP |
|--------|------|-------------|--------------|----------|-------------|


**Ejemplo cURL**
```bash
curl -X POST http://localhost:8000/ia_scrum_team/handoff \
  -H "Content-Type: application/json" \
  -H "X-API-KEY: $API_KEY" \
  -d '{"user_id":"u123","conversation_id":"c456","message":"Necesito ayuda con la factura","metadata":{"priority":"high"}}'
```

## 🔐 Seguridad
* **API‑Key**: Se valida contra la variable de entorno `API_KEY`. Si la clave no coincide, FastAPI devuelve `401 Unauthorized`.
* **CORS**: Sólo se permiten orígenes definidos en `ALLOWED_ORIGINS` (por defecto `*` en desarrollo).
* **Rate limiting**: Implementado mediante middleware de `slowapi` (límites configurables en `.env`).

## 📦 Deploy notes
* El contenedor expone el puerto **8000** y está registrado en Traefik con la regla `PathPrefix(`/ia_scrum_team`)`.
* Para que el **Kafka consumer** arranque, asegúrate de que la variable `KAFKA_BROKER` esté presente; de lo contrario el proceso se desactiva sin bloquear el servicio.
